# Tailscale 安裝闖關

一頁式互動教學，用闖關遊戲的方式，一步步帶才庫同事完成 Tailscale 下載、註冊、串接內網伺服器。

## 六道關卡

1. 任務簡報：為什麼要裝 Tailscale
2. 下載並安裝 Tailscale
3. 用 Google 帳號註冊登入（含「跳過」陷阱提醒）
4. 把 Email 填進回報表單送出
5. 接受內網裝置邀請
6. 在記事本回報「已完成」

## 特色

- 每關完成打勾，解鎖下一關，進度自動記在瀏覽器
- 動態箭頭與高亮框，指向關鍵按鈕
- 揭示文字動畫、通關彩帶特效
- 桌面與手機自適應

## 本機預覽

```bash
python -m http.server 8123
# 瀏覽器開 http://127.0.0.1:8123/
```

純靜態網頁（HTML / CSS / JS），無外部相依。
